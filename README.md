## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/dev-dami/go-chat.git
   cd go-chat
   ```

2. Initialize the Go module:
   ```sh
   go mod tidy
   ```

## Usage

To run the application, use the following command:

```sh
go run .
```
